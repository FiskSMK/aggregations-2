##Zadanie 1c Zmiana typu danych

Użyłem skryptu dla mongo shell
```js
db.train.find( { "tags" : { $type : 2 } } ).snapshot().forEach(
 function (x) {
  if (!Array.isArray(x.tags)){
   x.tags = x.tags.split('');
   db.train.save(x);
}});
```
